# Visionary-AI
The Project contain an AI-driven congnitive system which interprets real world scenes in realtime.
The Project is build in django and it is using Gemini api.<br>
<b>It has three Features<b/><br>
<b>1.History:<b/><br>
   It will the scene in internet and will provide you history of it.If it does not have real history it will generate fake history.<br>
<b>2.Future:<b/><br>
  It will predict the future based on history of scene.if it does not have real history then it will predict future by its own.<br>

<b>3.Present:<b/>
  It will explain the current scene with complete details.
